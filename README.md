AI-Powered Fall Detection System
**Overview**

This project is a Real-Time Fall Detection System developed using Deep Learning and Computer Vision techniques.
The system detects human falls from video streams using a custom-trained YOLO model and triggers emergency alerts when a fall is continuously detected.

**Features**
Real-time Fall Detection
YOLO-based Object Detection
Interactive Tkinter GUI
Emergency Alert System
Event Logging
Live FPS Monitoring
Detection Statistics Dashboard
Output Video Generation
Technologies Used
Python
YOLO (Ultralytics)
OpenCV
Tkinter
Deep Learning
Computer Vision
System Architecture
Load trained YOLO model
Read video stream
Detect human posture
Identify fall events
Trigger emergency alert
Generate annotated output video

Installation

**Clone the repository:**
git clone https://github.com/muhammadahmadcheema2004-oss/AI-Fall-Detection-System.git

**Move into project directory:**
cd AI-Fall-Detection-System

**Install dependencies:**
pip install -r requirements.txt

**Run:**
python main.py
